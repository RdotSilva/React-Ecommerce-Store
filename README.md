# React E-commerce store

E-commerce store built using React, Firebase, Redux, GraphQL.

## Installation

1. Install dependencies in main project folder.

```
npm install
```

2. Install dependencies in client folder.

```
npm install
```

## Running the servers

1. Start nodemon dev server.

```
npm run dev
```
